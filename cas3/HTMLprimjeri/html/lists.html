join(string)
Konvertuje sve elemente niza u string i nadovezuje ih u jedan string, odvojene datim argumentom string kao separatorom.

sort()
Sortira niz u mjestu, u alfabetskom poretku.
sort(function)
Sortira niz u mjestu. Argument function ima dva argumenta, i vraca negativan broj ako je njegov prvi argument "manji od" drugog, nulu, ako su jednaki i pozitivan broj ako je prvi "veci od" drugog.
